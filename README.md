# InkyPi-Direct-View
E-Ink Display simulator for InkyPi developers
## Features
- Simulates Spectra6 e-ink screen
- InkyPi setings applied
- Download simulated screenshot

## Installation
Download and install in any InkyPi directory.  I have mine in the / utils directory

Run the .py script and point your browser tab to port 5010 on whatever IP address youâ€™re running the Inky Pi developers mode on (Or http://127.0.0.1:5010)
